# Assets Directory

This directory should contain the following image files for the Teeth - Dental Journal landing page:

## Required Files:

1. **app-icon.webp** - The Teeth - Dental Journal app icon (recommended size: 1024x1024px)
2. **app-store-button.webp** - App Store download button badge
3. **screenshot-1.webp** - Screenshot showing dental records overview
4. **screenshot-2.webp** - Screenshot showing tooth tracking feature
5. **screenshot-3.webp** - Screenshot showing appointment reminders

## Image Specifications:

- **Format**: WebP (for optimal performance)
- **App Icon**: Square, 1024x1024px recommended, will be displayed with rounded corners
- **Screenshots**: iPhone screenshots in portrait orientation, recommended height ~2400px
- **App Store Button**: Standard App Store badge, approximately 200x60px

## Notes:

The HTML files reference these images with relative paths (e.g., `assets/app-icon.webp`).
Please add the actual image files to this directory for the website to display properly.
